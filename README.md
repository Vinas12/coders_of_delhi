
# Coders of Delhi

A Python-based social network analysis project that recommends **People You May Know** and **Pages You Might Like** using mutual connections and user relationship data.

# Features
- Data Cleaning & Preprocessing
- Friend Recommendation System
- Page Recommendation System
- JSON Data Analysis

# Technologies Used
- Python
- JSON
- Jupyter Notebook
